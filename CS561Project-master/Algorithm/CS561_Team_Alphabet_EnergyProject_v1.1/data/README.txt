This dataset contains sensor data that was collected in the home of
a volunteer adult resident and a dog.

The following activities are annotated within the dataset. 

Bed_to_toilet
Breakfast
Bed
C_work
Dinner
Laundry
Leave_home
Lunch
Night_wandering
R_medicine

The sensor events are generated from motion sensors (these sensor IDs begin
with "M") and temperature sensors (these sensor IDs begin with "T").
The layout of the sensors in the home is shown in the file Cairo.png.

You may download and analyze this data free of charge.  Please do not
distribute the data without explicit permission from the CASAS research group.
Please reference the CASAS group in any publications resulting from using
this dataset.
